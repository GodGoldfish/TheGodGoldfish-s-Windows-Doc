STEP 7 :

STEP 8 :
DESACTIVATE USELESS PERIPHERICALS 
PERIPHERICALS MANAGER -> system periphericals

----

STEP 9 :
(Open REGEDIT)
WINDOWS PERFORMANCE CONSUMPTION (how windows use the capacity of your computer)
HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Multimedia\SystemProfile
SystemResponsiveness -> 10
(if you are using multiple programs at the same time and not just one game i recommand you to put "10")

---

STEP 10 :
(Open REGEDIT)
(https://en.wikipedia.org/wiki/Prefetching)
PREFETCH :
HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Session Manager\Memory Management\PrefetchParameters
EnablePrefetcher -> 0
create DWORD 32 bits in the same directory -> EnableSuperfetch -> 0 

----

STEP 11 :
(Open REGEDIT)
(set the RAM capacity to reduce the "svchost.exe" in the background
HKEY_LOCAL_MACHINE\SYSTEM\ControlSet001\Control
SvcHostSplitThresholdInKB -> ? x 1024 x 1024
(THE CURRENT RAM IN GB THAT YOU ARE USING FOR EXEMPLE 32GB x 1024 x 1024 AND THEN WRITE AND ENTER THE "33554432" value)

----

STEP 12 :
(something useless but disabling HPET "High Precision Event Timer" removes the micro-stuttering and screen tearing that may occur during gameplay and render process)
(In CMD with admin permissions)

bcdedit /deletevalue useplatformclock
&
bcdedit /set disabledynamictick yes

----

STEP 13 :
DISABLE USELESS SERVICES THAT YOU ARE NOT USING / services.msc /
(FREE SOME RAM AND CPU UTILIZATION)
(In CMD with admin permissions)

SuperPrefetch :
sc config "SysMain" start= disabled

SCANNER :
sc config "stisvc" start= disabled

FAX :
sc config "Fax" start= disabled

NON-TABLET WINDOWS :
sc config "TabletInputService" start= disabled
